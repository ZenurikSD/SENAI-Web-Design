Posicionamento CSS

Elementos que aparecem um abaixo do outro são chamados de "bloco", ao contrário de elementos "inline", que aparecem um ao lado do outro, da mesma forma que cada palavra em um parágrafo.

É possível alterar a forma que os elementos são dispostos em uma página usando métodos como:

  • A propriedade display - Valores como block, inline ou inline-block podem mudar como os elementos se comportam na página.
  • Floats - Aplicar um valor float faz com que elementos "bloco" fiquem ao redor, 'flutuando' ao lado de outro elemento; tipo quando uma imagem aparece com texto em volta.
  • A propriedade position - Permite controlar precisamente a localização de caixas dentro de outras caixas, por exemplo.
  • Layout de tabela - Recursos feitos para estilizar tabelas HTML podem ser utilizados em elementos que não são tabelas, usando display: table e suas propriedades associadas.
  • Layout Multi-Coluna - Esse layout faz o conteúdo de um 'bloco' ser organizado em colunas, como em um jornal. Também é uma propriedade de display.
