Strings

Uma string é qualquer texto sem função (letras, palavras, frases etc.) envolto por aspas ou crases.

let cor = "Verde"
print(`A cor que você escolheu é ${cor}`)

// '1' é uma string. Não vai ser tratada como número.
// "var = 1" também é string. Não vai ser interpretada como variável.


Variáveis (expandir)

Variáveis permitem que você faça referência à mesma informação várias vezes. Em Javascript as variáveis podem ser definidas usando var e seu valor pode ser alterado chamando seu nome.

var exemplo = "criei uma variável do tipo string"
exemplo = 25 + 3    //Agora ela é um valor numérico


Constantes

Constantes recebem valores da mesma forma que variáveis, mas a diferença é que elas não podem ser reescritas nem ficar vazias.

Arrays

Um Array é um conjunto de valores representado dentro de colchetes [ ], tipo uma lista. Um array dentro de outro é conhecido como um "array aninhado" (nested array).

var ingredientes = ['farinha', 'leite', 'ovos', 'açúcar']


Eventos

Modificar elementos quando algo acontece. Por exemplo, quando o usuário clicar, tocar, passar o mouse em cima etc..
É qualquer atributo que começa com "on-":

Estrutura DOM (Document Object Model)

O Modelo de Objetos do Documento (ou DOM) é uma API que permite acessar e manipular o HTML e CSS utilizando comandos que o Javascript é capaz de entender.

Laços de repetição (loops)

"Existem vários tipos de loops, mas essencialmente todos eles fazem a mesma coisa: repetem uma ação um certo número de vezes"
Loops and iteration --MDN

for(){..}

O loop for vai se repetir até que uma condição especificada se torne falsa. Ele verifica a condição primeiro para depois decidir se vai executar a declaração.

for (expressão de início; condição; incremento){
   declaração }

do..while()

A declaração do..while vai se repetir até que uma condição especificada se torne falsa. A diferença aqui é que ele primeiro executa a declaração uma vez e depois testa a condição para saber se vai repetí-la.

do
  declaração
while (condição for verdadeira)

for(..in..){}

A declaração for..in executa iterações (repetições) a partir de uma variável especificada, percorrendo todas as propriedades de um objeto.
Para cada propriedade distinta, o Javascript executará uma iteração.

//para(variável em objeto){
//  declaração }

Daí acessamos o corpo do documento para anexar (append) um filho, que é o parágrafo com o texto, referenciado na variável prgrf. - Uma constante 'button' é criada para pegar todos os elementos do tipo botão; - Então usamos um laço de repetição (loop) do tipo for, que até agora eu não entendi o que faz. Cria um parágrafo que pode ser clicado para alterar parte do seu texto:

const prgrf = document.querySelector('p'); prgrf.addEventListener('click', atualizarNome); function atualizarNome() { let nome = prompt('Insira um novo nome'); para.textContent = 'Jogador 1:' + nome; } Seleciona-se o elemento com o método querySelector() para atribuir ele a uma constante, assim fica mais fácil chamar a constante ao invés do caminho completo toda vez. Adiciona um método "receptor de eventos" na constante para ouvir o sinal de quando um evento ocorre (um clique nesse caso) e responder enviando outra ação, que aqui é um bloco de código (função) chamada 'atualizarNome()': Por fim definese a função. Ela cria uma variável 'nome' que pergunta (prompt) ao usuário por um novo nome; e depois altera o conteúdo do texto na constante (prgrf) para incluir o nome que o usuário digitar.
