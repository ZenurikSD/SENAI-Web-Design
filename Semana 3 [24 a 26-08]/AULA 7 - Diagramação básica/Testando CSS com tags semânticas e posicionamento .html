Cada elemento tem um tipo de propriedade no seu posicionamento. Elementos que aparecem um abaixo do outro (como os itens da lista aí em baixo) são "blocks", e elementos que aparecem lado a lado (como cada palavra desse parágrafo) são "inline".

É possível alterar a forma que os elementos são dispostos em uma página usando métodos como:

  • A propriedade display - Valores como block, inline ou inline-block podem mudar como os elementos se comportam na página.
  • Floats - Aplicar um valor float faz com que elementos "bloco" fiquem ao redor, 'flutuando' ao lado de outro elemento; tipo quando uma imagem aparece com texto em volta.
  • A propriedade position - Permite controlar precisamente a localização de caixas dentro de outras caixas, por exemplo.
  • Layout table - Recursos feitos para estilizar tabelas HTML podem ser utilizados em elementos que não são tabelas, usando display: table e suas propriedades associadas.
  • Layout table-column - Esse layout faz o conteúdo de um 'bloco' ser organizado em colunas, como em um jornal. Também é uma propriedade de display.
