Introdução à Banco de Dados

Na minha definição, um banco de dados é uma estrutura de tabelas para guardar valores. Essas tabelas podem ter colunas compartilhadas entre si, daí o nome "banco de dados relacional".

Comandos comuns do SQL: 1 Criar o banco de dados e selecioná-lo CREATE DATABASE banco1; USE banco1; 2 Abrir uma nova tabela e definir as colunas com seu tipo de dado e argumentos opcionais: CREATE TABLE pessoas( ID tinyInt(20) NOT NULL AUTO_INCREMENT, nome varchar(30) NOT NULL UNIQUE, sexo enum('M','F') NOT NULL, altura decimal(3, 2), PRIMARY KEY (id) ); 3 Inserir dados na tabela em todos os campos ou em específicos INSERT INTO pessoas VALUES(1, 'Joãozin', 'M'); INSERT INTO pessoas(nome, sexo) VALUES('Alisson', 'M'); 4 Adicionar colunas à tabela: ALTER TABLE pessoas ADD nacionalidade varchar(30); 5 Exibir tudo da tabela ou colunas específicas: SELECT * FROM pessoa; SELECT nome FROM pessoa; NOT NULL diz que o campo não pode ficar sem valor. AUTO_INCREMENT Incrementa automaticamente um número. Valor MIN-imo ou MAX-imo : SELECT MIN(coluna) AS nome FROM tabela;
