Objetos

Objetos no Javascript são como objetos na vida real mesmo. Um carro por exemplo: carros vêm em várias formas e tamanhos, cores diferentes, marcas diferentes, etc. Todo carro tem essas propriedades, mas os valores são diferentes de um carro para o outro. Um Ford Focus vermelho e um Honda Civic azul são carros, mas de marcas, modelos e cores diferentes.
(Javascript - Objects)

No JS os objetos podem guardar propriedades e valores. Além disso, podemos definir uma propriedade como objeto por si só, ou seja, um objeto aninhado, um dentro do outro. Por exemplo:

const roupas = { camisetas: 'branca' }

//Acessando o objeto roupas, propriedade camisetas. A propriedade vira um objeto próprio.
roupas.camisetas = { camiseta-1: 'Azul', camiseta-2: 'Verde' }
