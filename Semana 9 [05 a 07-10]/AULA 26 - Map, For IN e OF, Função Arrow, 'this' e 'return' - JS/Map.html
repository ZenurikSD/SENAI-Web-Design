Javascript Map

Um map é um "tipo de coleção", assim como objetos: você guarda vários conjuntos de coisas diferentes. É uma coleção de elementos em que cada elemento é guardado como um par de (chave, valor). Pense como sendo a (propriedade: valor) em um objeto.

Mas diferente dos objetos, que só guardam símbolos ou strings, o map pode guardar qualquer tipo de dado, incluindo funções e outros objetos.
