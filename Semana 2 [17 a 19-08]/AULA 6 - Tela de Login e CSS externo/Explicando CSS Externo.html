Utilizando o atributo class

A classe serve para criar... classes. Grupos distintos de elementos do HTML para o CSS conseguir interpretar e distinguir.

Ou seja, para todos os "objetos" da classe 'título', o arquivo CSS vai estilizar com a cor verde.

Para chamar uma classe no CSS, usa-se ponto antes do nome
.titulo{color: green}
E para chamar uma id, usa-se o hash. (Nunca lembro o nome certo disso)
#h1{font-family: 'Segoe-UI'}

A ID, por ser única, sempre tem preferência à classe.
