Introdução ao CSS

Há 3 formas de trabalhar com o CSS:

Interno - As regras do CSS ficam dentro do HTML, no <head>, na forma da tag <style>.
Externo - O CSS é um arquivo separado, e você adiciona ao HTML fazendo uma referência para o arquivo, no <head> após a tag <title>.
Inline - Ele é feito na própria tag do HTML, como o atributo 'style'.

Trabalharemos com o CSS Inline por agora.
