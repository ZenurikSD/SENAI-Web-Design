Semântica

MDN Web Docs - Semantics

Em programação, semântica diz respeito ao significado da parte de um código - por exemplo "o que acontece ao executar aquela linha de JavaScript?", ou "qual propósito ou função daquele elemento do HTML?"

Semânticas no HTML

Por exemplo, no HTML, o elemento <h1> é um elemento semântico, que dá a função (ou significado) de "um título de primeiro nível" na sua página. E por padrão, os navegadores irão estilizar ele para ter a aparência de um título grande; mas com o CSS você pode usar outra "folha de estilo" para ter a aparência que você quiser.

Por outro lado, você pode fazer qualquer elemento parecer com um título grande, considere o seguinte:
<span style="font-size: 32px; margin: 21px 0;">Isso é um título H1?</span>

Isso irá renderizar como:
Isso é um título H1?

Certamente se parece com um, mas ele não possui valor semântico, então não terá nenhum dos benefícios do CSS e teria que ser estilizado individualmente.
