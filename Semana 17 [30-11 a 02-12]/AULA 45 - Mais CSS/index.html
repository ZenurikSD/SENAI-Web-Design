30/11 O que temos para hoje? - Box sizing

Por padrão é content-box e quando você altera padding, margin etc.. as caixas vão crescer, "ignorando" o tamanho definido. border-box define que essas mudanças serão feitas a partir da borda, então o tamanho é preservado.

Caixa 1 [150px] content-box Caixa 2 [150px] border-box

@media queries

Serve para mudar o estilo e conteúdo conforme o tamanho da tela, para ter um site adaptado à celular, tablet e desktop.

Mude o tamanho da janela para ver.
As convenções de tamanho são:

  • Igual ou menor que 600px é celular
  • Até 900 é tablet?
  • Maior é desktop

Mas o @media_query está em desuso, prefira fazer pelo Javascript.

innerWidth - JS

Também serve para alterar o CSS baseado no tamanho da tela. É a opção recomendada. Por agora precisa atualizar a página pra ver o resultado.
Verde - É menor ou igual a 700
