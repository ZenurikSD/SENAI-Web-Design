Arquivo-Mestre {Javascript}

A cada aula vou expandir no que puder explicando os conceitos do JS e programação

O Javascript é a linguagem de programação usada para programar o comportamento e adicionar funções às páginas web. Existem vários termos estranhos que é preciso se familiarizar primeiro para entender a linguagem; mas você vai ver que com a perspectiva certa, não é nada difícil.

"O Javascript é composto de métodos que alteram as funções, classes e objetos"

Exercício semáforo

Clicar "acende" e tirar o mouse "apaga" (recarrega a página, o que é meio cursed, mas não achei um método melhor).

Eventos

Modificar elementos quando algo acontece. Por exemplo, quando o usuário clicar, tocar, passar o mouse em cima etc..
É qualquer atributo que começa com "on-":

onmouseover Passa o mouse aqui

Estrutura DOM (Document Object Model)

O DOM é responsável por "traduzir" os elementos HTML em objetos e propriedades que o Javascript é capaz de interpretar, permitindo modificá-los.

Funções!

Apesar do nome assustar um pouco, funções são blocos de código isolados, que só executam quando são chamadas

Sintaxe:
function Declara que è uma função. Seguido do nome da função, você usa parênteses para definir argumentos, mas no início ela vai estar vazia mesmo
